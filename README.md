<a id="korean"></a>
## 🇰🇷 한국어

**Librarius**는 Rust와 Tauri v2를 기반으로 구축된 고성능 데스크톱 웹 아카이빙 도구입니다. Chromium CDP를 활용한 정밀한 브라우저 제어와 SurrealDB를 이용한 로컬 데이터 영속성을 제공합니다.

### 주요 기능
* **스마트 URL 정규화:** 리다이렉션을 추적하여 최종 목적지 저장
* **실시간 모니터링:** 헤드리스 브라우저 화면의 실시간 뷰 제공
* **재귀적 BFS 크롤러:** 깊이 기반 링크 탐색
* **PDF 변환:** 수집된 데이터를 Typst 엔진을 통해 책 형태의 PDF로 생성

<a id="english"></a>
## 🇺🇸 English

**Librarius** is a high-performance desktop web archiving tool built with Rust and Tauri v2. It offers precise browser control using Chromium CDP and local data persistence via SurrealDB.

### Key Features
* **Smart URL Normalization:** Tracks redirections to store the final destination.
* **Real-time Monitoring:** Provides a live view of the headless browser.
* **Recursive BFS Crawler:** Depth-based link exploration.
* **PDF Conversion:** Generates book-style PDFs from collected data using the Typst engine.

## 🇨🇳 简体中文

**Librarius** 是一款基于 Rust 和 Tauri v2 构建的高性能桌面网页存档工具。它利用 Chromium CDP 实现精确的浏览器控制，并通过 SurrealDB 提供本地数据持久化。

### 主要功能
* **智能 URL 正规化：** 追踪重定向以保存最终目的地
* **实时监控：** 提供无头浏览器画面的实时视图
* **递归 BFS 爬虫：** 基于深度的链接探索
* **PDF 转换：** 使用 Typst 引擎将收集的数据生成书籍形式的 PDF
